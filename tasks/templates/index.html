Este es un proyecto de una tienda de ropa en línea que ofrece una experiencia de compra simplificada y personalizada. Los visitantes pueden elegir entre ropa para hombre o mujer, así como tipos de prendas con un solo clic. El carrito de compras mantiene un seguimiento en tiempo real del total de artículos seleccionados, brindando transparencia en el proceso. Además, los usuarios pueden registrarse y tener sus propias cuentas para una experiencia más personalizada. Todo esto fue posible gracias al uso de las tecnologías de Django, JavaScript, npm, Gulp y Sass. Es tu destino ideal para descubrir moda de manera eficiente y elegante.

Herramientas que se ocuparon para este proyecto

BienesRaices es tu puerta de entrada al emocionante mundo de la compra y venta de propiedades. En esta plataforma, el administrador tiene el control total, con la capacidad de listar propiedades y crear vendedores, todo con un diseño intuitivo y fácil de usar. Los vendedores también pueden gestionar sus propias ofertas y propiedades a través de un panel de control. Este proyecto se ha desarrollado utilizando PHP con el paradigma de "Active Record", junto con JS, npm, Gulp y Sass para una experiencia de usuario fluida y atractiva. ¿Quieres analizar el código por ti mismo? ¡Puedes encontrar el proyecto en mi GitHub! Descubre la comodidad y la eficiencia de BienesRaices y comienza tu viaje en bienes raíces hoy.

Herramientas que se ocuparon para este proyecto

"Sheru Website" es una creación artística y un destino de compras en uno. Este sitio web es el escaparate personal de un talentoso artista independiente, que comparte sus canciones con el mundo y ofrece una visión única de la inspiración detrás de cada melodía. Además de su faceta musical, este proyecto incluye una tienda en línea donde el artista puede vender su mercancía, y todo es gestionado desde un panel de control de administrador intuitivo que permite manipular noticias, usuarios, productos y más.


El diseño responsive garantiza una experiencia perfecta en cualquier dispositivo, y la implementación de tecnologías como SASS, Gulp, NPM, PHP, JS y Node.js aseguran un rendimiento excepcional. Además, un scroll horizontal adaptable a dispositivos móviles brinda un toque único.


¿Te gustaría analizar el código por ti mismo? ¡Puedes encontrar este emocionante proyecto en mi GitHub! Explora la creatividad y la versatilidad de Sheru Website para descubrir la música y el arte con un giro personal.
